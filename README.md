# web4you

Modern website selling platform built with HTML, CSS, Bootstrap, JavaScript/jQuery (AJAX), PHP and MySQL.

## Setup
1. Import `database.sql` into your MySQL database.
2. Upload all files to your hosting (InfinityFree-compatible).
3. Update `config.php` if needed.
4. Open `index.html`.
